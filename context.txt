  Workflow – login

  1. User types email / password, clicks Continue.
  2.  Login.jsx  →  login(email,password)  (Firebase Auth).
  3. Firebase returns user object;  onAuthStateChanged  updates  currentUser  in context.
  4.  Login.jsx   navigate('/dashboard') .
  5.  Dashboard.jsx  runs  currentUser.getIdToken()  → JWT.
  6. Axios sends GET  http://localhost:8080/api/dashboard  with header  Authorization: Bearer <jwt> .

  Dashboard render

  •  useEffect  stores response in  data .
  • KPI cards read  data.kpis.* .
  • Recharts  <AreaChart>  plots  data.salesAnalytics .
  • Table loops  data.invoices .
  • Logout button calls  logout()  → Firebase clears token,  navigate('/login') .

  Workflow – token verification

  1. Request arrives at  /api/dashboard  with  Authorization: Bearer <jwt> .
  2. Controller extracts token (strip  "Bearer " ).
  3.  FirebaseAuth.verifyIdToken(token)  → ensures token signed by your Firebase project, not expired.
  4. If verification fails →  401 UNAUTHORIZED .
  5. If passes → assemble static map:

    {
      "kpis": { "orderQuantity": {...}, "avgOrderValue": {...}, "impressions": {...} },
      "salesAnalytics": [{ "name": "Jan", "value": 2900 }, ...],
      "invoices": [{ "id": "#065499", "customerName": "Makima Waifu", ... }, ...]
    }

  6.  ResponseEntity.ok(response)  returns JSON; React consumes it.
